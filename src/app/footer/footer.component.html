Este programa de contabilidad fue desarrollado como parte de un proyecto académico en el Instituto Paula Robles, con el objetivo de ofrecer una solución profesional, eficiente y accesible para la gestión contable de pequeñas y medianas empresas. El sistema está diseñado para facilitar el registro, control y análisis de operaciones contables, contribuyendo a una administración financiera más organizada y transparente.
